## Logistics Management System Multi-identity

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/109cf24bcbb14634ad5758231eb5dd23/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User (Customer), Employee, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Logistics Information Management
6. Vehicle Management
7. Freight Management



**Employee Functions:**
1. Login
2. Change Password
3. Personal Information
4. Logistics Information Management
5. Driver Management
6. Vehicle Management
7. Freight Management
8. Customer Management
9. Employee Notification Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (Vehicle Type Management, Logistics Information Type, Freight Type Management, Notification Type Management, Item Type Management)
5. Logistics Information Management
6. Driver Management
7. Vehicle Management
8. Freight Management
9. Customer Management
10. Notice Board Management
11. Employee Management
